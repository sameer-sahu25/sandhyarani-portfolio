# Sandhyarani Pursi — Portfolio

Portfolio of Sandhyarani Pursi, an aspiring product designer passionate about visual storytelling and solving real-world problems through design.

## Features

- **Modern UI**: Built with React, Tailwind CSS, and Shadcn UI.
- **Animations**: Powered by Framer Motion.
- **Icons**: Lucide React.
- **Form Handling**: React Hook Form and Zod validation.
- **State Management**: React Query.
- **Testing**: Vitest for unit testing and Playwright for E2E.

## Getting Started

1.  **Install dependencies**:
    ```bash
    npm install
    ```

2.  **Run development server**:
    ```bash
    npm run dev
    ```

3.  **Build for production**:
    ```bash
    npm run build
    ```

4.  **Run tests**:
    ```bash
    npm run test
    ```
